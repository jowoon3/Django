CSS 리스트(List)

CSS를 이용하면 리스트에 다양한 효과를 적용할 수 있다.

  1. 순서가 없는 리스트(unordered list)
  2. 순서가 있는 리스트(ordered list)
  3. 정의 리스트(definition list)

CSS에서 사용할 수 있는 list-style 속성

1. list-style-type
2. list-style-image
3. list-style-position

list-style-type 속성

리스트 요소의 앞에 위치하는 숫자나 기호를 마커(marker)라고 한다.
list-style-type 속성을 이용하면 리스트에 다양한 마커(marker)를 적용할 수 있다.

html 리스트 마커

예제

  • 수박
  • 참외
  • 사과
  • 멜론
  1. An Apple is good
  2. I like Melon
  3. We bought a Banana
  4. This corn soup tastes good

<style>
.circle { list-style-type: circle; }
 .square { list-style-type: square; }
 .upperAlpha { list-style-type: upper-alpha; }
 .lowerRoman { list-style-type: lower-roman; }
</style>

list-style-image 속성

list-style-image 속성을 이용하면 마커(marker)로 자신만의 이미지를 사용할 수 있다.

예제

<style>
 .imageMarker { list-style-image: url("/examples/images/img_list_marker.png"); }
</style>

list-style-position 속성

list-style-position 속성을 이용하면 리스트 요소의 위치를 설정할 수 있다.
list-style-position 속성의 기본 속성값은 outside로 설정되어 있다. (아웃사이드가 눈에 이뻐보임)

예제

<style>
 .outside { list-style-position: outside; }
 .inside { list-style-position: inside; }
</style>

list-style 속성 한 번에 적용하기

위에서 언급한 모든 list-style 속성을 이용한 스타일을 한 줄에 설정할 수 있다.

예제

<style>
 ul { list-style: square inside url("/examples/images/img_list_marker.png"); }
</style>

리스트에 배경색 적용

CSS를 사용하면 리스트 전체뿐만 아니라 리스트 요소별로도 각각의 배경색을 설정할 수 있다.

예제

    여름에 먹는 과일
  • 수박
  • 참외
  • 사과
  • 멜론
스타일은 div 외부에서 설정해야 함
<style>
 ul { background: #D2691E; padding: 15px; }
 ol { background: #6495ED; padding: 15px; }
 ul li { background: #DEB887; margin: 5px; }
 ol li { background: #00FFFF; margin-left: 15px; }
</style>

CSS list-style 속성

속성 설명
list-style 모든 list-style 속성을 이용한 스타일을 한 줄에 설정할 수 있음.
list-style-type 리스트 요소의 마커(marker)를 설정함.
list-style-image 리스트 요소의 마커로 사용할 이미지를 설정함.
list-style-position 리스트 요소의 위치를 설정함.

CSS 테이블(Table)

CSS를 이용하면 다양한 형태의 테이블을 만들 수 있다.

border border-collapse border-spacing
caption-side empty-cells table-layout

테이블에 다음 속성을 사용하여 CSS 스타일을 적용할 수 있다.
1. border
2. border-collapse
3. border-spacing
4. caption-side
5. empty-cells
6. table-layout

border 속성

border 속성으로 테이블의 테두리(border)를 설정할 수 있다.
이 속성을 명시하지 않으면 해당 테이블은 기본 설정으로 빈 테두리를 가지게 된다.

테이블예제

예제에서 테이블의 테두리(border)가 두 줄씩 나타나는 이유는 <th>태그와 <td>태그가 각각 자신만의 테두리를 가지고 있기 때문
위와 같이 두 줄로 표현되는 테두리를 한 줄로 설정하려면 border-collapse 속성을 사용해야 한다.

border-collapse 속성

border-collapse 속성값을 collapse로 설정하면 해당 테이블의 테두리는 한 줄로 표현된다.
이 속성을 명시하지 않으면 해당 테이블은 기본 설정으로 테이블 요소별 테두리를 모두 표현하게 된다.

예제

테이블예제

border-spacing 속성

border-spacing 속성은 테이블 요소(th, td)간의 여백을 설정해 준다

예제

border border-collapse border-spacing
caption-side empty-cells table-layout

<style>

 { width: 50%; border:2px solid #3b6c9b}
table {border-collapse: seperate; border-spacing: 5px 10px; font-weight: bold; font-size: 15px; text-align: center}
table>thead>tr {background-color: lightgray; height: 50px; border-bottom: 2px solid #CD5C5C; padding: 10p}
table>tbody>tr {width: 33%;}
</style>

text-align 속성

text-align 속성은 테이블 요소(th, td) 내부에서 텍스트의 수평 방향 정렬을 설정한다.
<th>태그 내부는 가운데 정렬이, <td>태그 내부는 왼쪽 정렬이 기본 설정

예제

테이블예제
텍스트얼라인
<style>

table, th, td { border: 1px solid black; }
table { border-collapse: collapse; width: 100%; }
th { text-align: left; }
td { text-align: center; }
</style>

vertical-align 속성

vertical-align 속성은 테이블 요소(th, td) 내부에서 텍스트의 수직 방향 정렬을 설정한다.
<th>태그와 <td>태그 모두 가운데 정렬이 기본 설정

예제

테이블예제
버티컬얼라인
<style>
table, th, td { border: 1px solid black; }
table { border-collapse: collapse; width: 100%; }
th { vertical-align: top; height: 50px; }
td { vertical-align: bottom; height: 50px; }
</style>

다양한 형태의 테이블 예제

CSS를 이용하면 HTML 기본 테이블을 훨씬 더 다양한 모습으로 설정할 수 있다.

1. <th>태그와 <td>태그에 border-bottom 속성을 사용하여 수평 나눔선만으로 만든 테이블

테이블예제
수평나눔선만
<style>
table { border-collapse: collapse; width: 100%; }
th, td { padding: 10px; border-bottom: 1px solid #CD5C5C; }
</style>

2. :hover 선택자를 이용하여 <tr>태그에 마우스를 올리면 행 전체가 하이라이트 되도록 만든 테이블
테이블예제
hover 이용
<style>
table { border-collapse: collapse; width: 100%; }
th, td { padding: 10px; border-bottom: 1px solid #CD5C5C; }
tr:hover { background-color: #F5F5F5; }

</style>

:hover 는 CSS 동적 의사 클래스에서 더 자세히 다룬다

3. background-color 속성과 :nth-child 선택자를 사용하여 얼룩무늬 효과를 설정한 테이블

여름 과일
수박
참외
딸기
포도
<style>
table { border-collapse: collapse; width: 100%; }
th, td { padding: 10px; }
tr:nth-child(odd) { background-color: #F3F3F3; }
</style>

:nth-child 는 CSS 구조 의사 클래스에서 자세히 다룬다.

CSS table 속성

속성 설명
border 모든 border 속성을 이용한 스타일을 한 줄에 설정할 수 있음.
border-collapse 테이블의 테두리를 한 줄로 나타낼지를 설정함.
border-spacing 테이블 요소(th, td)간의 여백을 설정함.
caption-side 테이블의 캡션(caption)을 설정함.
empty-cells 테이블 내의 빈 셀(cell)들의 테두리 및 배경색을 표현할지 안 할지를 설정함.
table-layout 테이블에 사용되는 레이아웃 알고리즘을 설정함.
