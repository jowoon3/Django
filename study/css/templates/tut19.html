box-shadow 속성

box-shadow 속성의 문법은 text-shadow 속성을 사용하는 문법과 같다.

box-shadow: [그림자의x축값 음수면 왼쪽 양수면 오른쪽] [그림자의y축값 음수면 위쪽 양수면 아래쪽] [blur값 (흐릿함정도)] [색상];

box-shadow : inset 이라고 시작하면 그림자가 박스의 안쪽으로 채워진다

플렉스 박스(flex Box)의 개념

플렉스 박스(flex Box)는 플렉스 컨테이너(flex container)와 플렉스 요소(flex item)로 구성된다.
플렉스 컨테이너(flex container)는 해당 HTML 요소의 display 속성을 설정하는 것
해당 요소를 블록 타입으로 정의하려면 display 속성값을 flex로, 인라인 타입으로 정의하려면 inline-flex로 설정한다.
플렉스 컨테이너는 언제나 하나 이상의 플렉스 아이템을 포함해야 한다.
플렉스 컨테이너의 외부와 플렉스 아이템의 내부의 모든 것들은 평소처럼 동작한다.
플렉스 박스(flex Box)는 오직 플렉스 아이템이 플렉스 컨테이너의 내부에서 어떻게 위치하는가만을 정의한다.
플렉스 아이템은 플렉스 컨테이너 안에서 플렉스 라인(flex line)이라는 가상의 선을 따라 위치하게 된다.
기본적으로 하나의 플렉스 컨테이너는 오직 단 하나의 플렉스 라인만을 가지고 있다.

direction 속성을 이용하면 이러한 플렉스 라인(flex line)의 방향을 바꿀 수도 있다.
direction 속성값이 rtl(right-to-left)로 설정되면, 페이지 내의 모든 텍스트 요소는 오른쪽에서 왼쪽 방향으로 써진다.
그와 동시에 플렉스 라인(flex line)의 방향도 바뀌게 되어, 플렉스 아이템의 정렬도 오른쪽에서 왼쪽으로 바뀌게 된다.

flex-direction 속성

flex-direction 속성은 플렉스 컨테이너 안에서 플렉스 아이템이 배치될 방향을 설정한다.

1. row : 기본 설정으로, 플렉스 아이템은 왼쪽에서 오른쪽으로, 그리고 위쪽에서 아래쪽으로 배치된다.
2. row-reverse : 만약에 direction 속성값이 ltr(left-to-right)이면, 플렉스 아이템은 반대로 오른쪽에서 왼쪽으로 배치된다.
3. column : 만약에 쓰기 방식이 수평이면, 플렉스 아이템은 수직 방향으로 위쪽에서 아래쪽으로 배치된다.
4. column-reverse : 만약에 쓰기 방식이 수평이면, 플렉스 아이템은 수직 방향으로 아래쪽에서 위쪽으로 배치된다.

justify-content 속성

justify-content 속성은 플렉스 아이템의 수평 방향 정렬 방식을 설정한다.
이 속성은 다음과 같은 속성값을 가질 수 있다.
1. flex-start : 기본 설정으로, 플렉스 아이템은 플렉스 컨테이너의 앞쪽에서부터 배치된다.
2. flex-end : 플렉스 아이템은 플렉스 컨테이너의 뒤쪽에서부터 배치된다.
3. center : 플렉스 아이템은 플렉스 컨테이너의 가운데에서부터 배치된다.
4. space-between : 플렉스 아이템은 아이템들 사이에만 여유 공간을 두고 배치된다.
5. space-around : 플렉스 아이템은 앞, 뒤, 그리고 아이템들 사이에도 모두 여유 공간을 두고 배치된다.

align-items 속성

align-items 속성은 플렉스 아이템의 수직 방향 정렬 방식을 설정한다.
이 속성은 한 줄만을 가지는 플렉스 박스에서는 효과가 없으며, 두 줄 이상을 가지는 플렉스 박스에서만 효과가 있다.
이 속성은 다음과 같은 속성값을 가질 수 있다.
1. stretch : 기본 설정으로, 플렉스 아이템의 높이가 플렉스 컨테이너의 높이와 같게 변경된 뒤 연이어 배치된다.
2. flex-start : 플렉스 아이템은 플렉스 컨테이너의 위쪽에 배치된다.
3. flex-end : 플렉스 아이템은 플렉스 컨테이너의 아래쪽에 배치된다.
4. center : 플렉스 아이템은 플렉스 컨테이너의 가운데에 배치된다.
5. baseline : 플렉스 아이템은 플렉스 컨테이너의 기준선(baseline)에 배치된다.

flex-wrap 속성 (줄넘김 처리)


flex-wrap 속성은 플렉스 라인에 더 이상의 여유 공간이 없을 때, 플렉스 아이템의 위치를 다음 줄로 넘길지를 설정한다.
이 속성은 다음과 같은 속성값을 가질 수 있다.

align-content 속성

align-content 속성은 flex-wrap 속성의 동작을 변경할 수 있다.
flex-wrap: wrap;이 설정된 상태에서, 아이템들의 행이 2줄 이상 되었을 때의 수직축 방향 정렬을 결정하는 속성이다
align-items 속성과 비슷한 동작을 하지만, 플렉스 아이템을 정렬하는 대신에 플렉스 라인을 정렬한다.
align-content 속성
1. stretch : 기본 설정으로, 플렉스 라인의 높이가 남는 공간을 전부 차지하게 된다.
2. flex-start : 플렉스 라인은 플렉스 컨테이너의 앞쪽에 뭉치게 된다.
3. flex-end : 플렉스 라인은 플렉스 컨테이너의 뒤쪽에 뭉치게 된다.
4. center : 플렉스 라인은 플렉스 컨테이너의 가운데에 뭉치게 된다.
5. space-between : 플렉스 라인은 플렉스 컨테이너에 고르게 분포된다.
6. space-around : 플렉스 라인은 플렉스 컨테이너에 고르게 분포된다. 단, 양쪽 끝에 약간의 공간을 남겨둔다.

플렉스 요소(flex item)의 flex 속성

flex 속성을 이용하면 같은 플렉스 컨테이너 안에 있는 플렉스 아이템의 너비를 상대적으로 설정할 수 있다.

flow-grow 는 남는 행 여백을 분배해서 채우는 방법이다. flex-grow 속성이 적용되지 않거나, 속성 값이 "0" 인 경우
레이아웃 너비보다 아이템들의 너비 합이 더 작으면 아이템 오른쪽 끝에 여백이 남게 된다.
이 속성을 적용하면 아이템들이 주어진 flow-grow로 설정한 비율대로 늘어나면서 레이아웃 영역을 채우게 된다.

flow-shrink 는 레이아웃을 벗어난 아이템 너비를 분배해서 줄이는 방법이다.
flex-shrink는 "flex-wrap: wrap;" 속성을 부여한 경우 적용되지 않으므로 주의한다.
flex-wrap 속성을 정의하지 않거나 "flex-wrap: nowrap;" 속성을 부여하면
flex-shrink 속성은 기본 값이 "1"로 되어있어 자동으로 아이템이 축소되어 적용된다.
그러므로 자동으로 아이템 너비가 축소되지 않으려면 반드시 "flex-shrink: 0;"이라고 선언해야 한다.

flex-basis는 Flex 아이템의 기본 크기를 설정한다. (flex-direction이 row일 때는 너비, column일 때는 높이)

flex는 flex-grow, flex-shrink, flex-basis를 한 번에 쓸 수 있는 축약형 속성이다.
예시 flex: 1 => flex-grow: 1; flex-shrink: 1; flex-basis: 0%;
flex: 1 1 auto; => flex-grow: 1; flex-shrink: 1; flex-basis: auto;
flex: 1 500px; => flex-grow: 1; flex-shrink: 1; flex-basis: 500px;

flex-flow

flex-direction 속성과 flex-wrap 속성을 이용한 스타일을 한 줄에 설정할 수 있음.
align-content flex-wrap 속성의 동작을 변경할 수 있으며, 플렉스 아이템을 정렬하는 대신에 플렉스 라인을 정렬함.
order 플렉스 컨테이너 안에 있는 플렉스 아이템들의 순서를 설정함.
align-self

플렉스 아이템마다 서로 다른 align 속성값을 설정함.
flex

같은 플렉스 컨테이너 안에 있는 플렉스 아이템의 너비를 상대적으로 설정함.

단, IE10 이하의 브라우저에서는 지원하지 않는다.

align-content / justify-content

flexbox처럼 세로축으로 정렬하는 align-content와 가로축으로 정렬하는 justify-content까지 기능이 같다.
값도 start, center, end, space-around, space-between, space-evenly, stretch로 똑같다.

minmax()

column, row의 최소 / 최대 크기를 지정한다. 인자 두 개가 필요한데, 각각 min값 / max값을 입력한다.
이 값을 지정해 두면 해당 크기보다 작아지거나 커지지 않는다. (min-width, max-width랑 비슷!)

auto-fill

column, row의 개수를 컨테이너에 맞게 자동적으로 조절해 주어 쉽게 반응형을 만들 수 있다.
repeat() 함수와 minmax() 함수랑 잘 어울린다. repeat(auto-fill, minmax(150px, 1fr))
