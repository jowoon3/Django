Button이 아닌것을 버튼으로 만들고 싶을때 Button tags

Popovers



popovers 기본형태


popovers 위치


popovers custom container

parent 요소의 어떤 스타일 때문에 popover가 방해되는 경우. popover가 특정 element 내부에서만 작용하도록 container 형식을 특정하고 싶을때가 있다.
<예1>
const popover = new bootstrap.Popover('.example-popover', {
container: 'body'
})

<예2>
const popover = new bootstrap.Popover('.example-popover', {
container: '.modal-body'
})

custom popover style을 수정해서 덮어씌울수 있다.
<예3>
.custom-popover {
--bs-popover-max-width: 200px;
--bs-popover-border-color: var(--bs-primary);
--bs-popover-header-bg: var(--bs-primary);
--bs-popover-header-color: var(--bs-white);
--bs-popover-body-padding-x: 1rem;
--bs-popover-body-padding-y: .5rem;
}


popovers 다음 클릭에서 해제되는 것
