CSS3 색(Color) 표현

CSS3에서 새롭게 추가된 색 표현 방법

1. RGBA 색상값으로 표현

2. HSL 색상값으로 표현

3. HSLA 색상값으로 표현

4. opacity 속성

CSS2 색

CSS2까지는 색을 표현하기 위해서 다음 방법만을 사용할 수 있었다.
1. 색상 이름으로 표현
2. RGB 색상값으로 표현
3. 16진수 색상값으로 표현

CSS3에서 추가된 색상값 및 opacity 속성 지원 버전

CSS3에서 추가된 색상값 및 opacity 속성을 지원하는 주요 웹 브라우저의 버전

속성

RGBA 색상값,

HSL 색상값,

HSLA 색상값

 9.0 4.0 3.0 3.1 10.1
opacity 9.0 4.0 2.0 3.1 10.1

RGBA 색상값으로 표현

RGBA 색상값은 RGB 색상값에 알파 채널 값을 더한 색상값이다.
알파 채널(alpha channel)이란 색상의 투명도를 나타내는 채널이다.
알파 채널 값은 완전한 투명 상태인 0.0부터 투명도가 전혀 없는 1.0 사이의 값을 가진다.

예제

header_01{background-color: rgba(0,255,0,0);}

header_02{background-color: rgba(0,255,0,0);}

header_03{background-color: rgba(0,255,0,0.4);}

header_04{background-color: rgba(0,255,0,0.6);}

header_05{background-color: rgba(0,255,0,0.8);}

header_06{background-color: rgba(0,255,0,1);}

HSL 색상값으로 표현

HSL 색상값은 빛의 삼원색으로 색을 표현하는 RGB 색상값과는 달리 색상, 채도, 명도를 사용해서 색을 표현한다.
HSL 색상값에서 HSL은 각각 색상(Hue), 채도(Saturation), 명도(Lightness)를 의미한다.

색상은 0부터 360 사이의 값을 가지며, 색상환(color wheel)의 각도를 나타낸다.

색상 값이 0 또는 360이면 빨간색(red)이 되며, 120이면 녹색(green), 240이면 파란색(blue)이 된다.

예제

header_07{background-color: hsl(0, 100%, 50%);}

header_08{background-color: hsl(90, 100%, 50%);}

header_09{background-color: hsl(180, 100%, 50%);}

header_10{background-color: hsl(270, 100%, 50%);}

header_11{background-color: hsl(360, 100%, 50%);}

채도는 0%부터 100% 사이의 값을 가지며, 색상의 연하고 짙은 정도를 나타낸다.
채도 값이 0%면 회색이 되고, 100%면 원래 색상이 된다.

예제

header_12{background-color: hsl(0, 0%, 50%);}

header_13{background-color: hsl(0, 20%, 50%);}

header_14 {background-color: hsl(0, 40%, 50%);}

header_15 {background-color: hsl(0, 60%, 50%);}

header_16{background-color: hsl(0, 80%, 50%);}

header_17{background-color: hsl(0, 100%, 50%);}

명도는 0%부터 100% 사이의 값을 가지며, 색상의 밝고 어두운 정도를 나타낸다.
명도 값이 0%면 검정색이 되고, 50%면 원래 색상, 100%면 흰색이 된다.

예제

header_18 {background-color: hsl(0, 100%, 0%);}

header_19 {background-color: hsl(0, 100%, 20%);}

header_20 {background-color: hsl(0, 100%, 40%);}

header_21 {background-color: hsl(0, 100%, 50%);}

header_22 {background-color: hsl(0, 100%, 60%);}

header_23 {background-color: hsl(0, 100%, 80%);}

header_24 {background-color: hsl(0, 100%, 100%);}

HSLA 색상값으로 표현

HSLA 색상값은 HSL 색상값에 알파 채널 값을 더한 색상값이다.

예제

header_25 {background-color: hsla(0, 100%, 50%, 0);}

header_26 {background-color: hsla(0, 100%, 50%, 0.2);}

header_27 {background-color: hsla(0, 100%, 50%, 0.4);}

header_28 {background-color: hsla(0, 100%, 50%, 0.6);}

header_29 {background-color: hsla(0, 100%, 50%, 0.8);

header_30 {background-color: hsla(0, 100%, 50%, 1);}

opacity 속성

opacity 속성은 색상에 대한 투명도를 설정해 준다.
opacity 속성값은 완전한 투명 상태인 0.0부터 투명도가 전혀 없는 1.0 사이의 값을 가진다.

예제

header_31 {background-color: rgb(0,255,0); opacity:0}

header_32 {background-color: rgb(0,255,0); opacity:0.2}

header_33 {background-color: rgb(0,255,0); opacity:0.4}

header_34 {background-color: rgb(0,255,0); opacity:0.6}

header_35 {background-color: rgb(0,255,0); opacity:0.8}

header_36 {background-color: rgb(0,255,0); opacity:1}

opacity 속성과 알파 채널과의 차이점

위에서 살펴본 opacity 속성과 알파 채널 모두 투명도를 조절한다는 공통점을 가지고 있다.
opacity 속성은 투명도를 설정했을 때 설정한 요소의 모든 자식(child) 요소까지 전부 같은 투명도로 설정한다.
하지만 알파 채널은 투명도를 설정한 요소에만 투명도를 설정하는 차이점이 존재한다.
다음 예제는 opacity 속성과 알파 채널과의 차이점을 보여주는 예제이다.

예제

para_01 {background-color: rgb(255,0,0); opacity:0}

para_06 {background-color: rgb(255,0,0); opacity:1}

para_07 {background-color: rgba(255,0,0,0);}

para_12 {background-color: rgba(255,0,0,1);}

CSS 배경(Background)

웹 페이지뿐만 아니라 HTML 요소는 모두 각자의 배경을 가지고 있다.
CSS의 background 속성은 이러한 각 요소의 배경에 다양한 효과를 줄 수 있게 해준다

CSS에서 사용할 수 있는 background 속성은 다음과 같다.
1. background-color
2. background-image
3. background-repeat
4. background-position
5. background-attachment

background-color 속성

background-color 속성은 해당 HTML 요소의 배경색(background color)을 설정한다.

예제

<style>
body { background-color: lightblue; }
h4 { background-color: rgb(255,128,0); }
>p { background-color: #FFFFCC; }
</style>

background-image 속성

background-image 속성은 해당 HTML 요소의 배경으로 나타날 배경 이미지(image)를 설정한다.
설정된 배경 이미지는 기본 설정으로 HTML 요소 전체에 걸쳐 반복되어 나타난다.
배경 이미지를 사용할 때에는 이미지가 본문의 텍스트를 방해하지 않도록 주의를 기울여야 한다.

나쁜예

<style>
body { background-image: url("http://www.tcpschool.com/examples/images/img_background_bad.png"); }
</style>

좋은예

<style>
body { background-image: url("http://www.tcpschool.com/examples/images/img_background_good.png"); }
</style>

background-repeat 속성

배경 이미지는 기본 설정으로 수평과 수직 방향으로 모두 반복되어 나타난다.

background-repeat 속성을 이용하면 이러한 배경 이미지를 수평이나 수직 방향으로만 반복되도록 설정할 수 있다.

배경이미지 수평 반복 예제

<style>
body { background-image: url("http://www.tcpschool.com/examples/images/img_man.png"); background-repeat: repeat-x; }
</style>

배경 이미지 수직 반복 예제

<style>
body { background-image: url("http://www.tcpschool.com/examples/images/img_background_bad.png"); background-repeat: repeat-y; }
</style>

배경 이미지가 반복되지 않고 한 번만 나타나게 할 수도 있다.
백그라운드-반복 속성에서 repat-x 는 x 축으로만 반복하고 repeat-y는 y축으로만 반복한다

배경이미지 한번만 나타나는 예제

<style>
body { background-image: url("http://www.tcpschool.com/examples/images/img_man.png"); background-repeat: no-repeat; }
</style>

배경이미지가 랜덤하게 변경될 경우, 배경이미지의 명암에 따라 글씨가 선명하거나 흐리게 보일수 있다
이경우 배경이미지의 투명도를 0.5로 낮추면 배경이미지가 변경되어도 글씨의 선명함을 유지할수 있다.
배경이미지 삽입후 아예 글씨가 안보이면 배경이미지를 맨뒤로 보내기 위해 z-index :-1 이라고 설정해준다.

background-position 속성

background-position 속성은 반복되지 않는 배경 이미지의 상대 위치(relative position)를 설정한다.

예제

<style>
body{
background-image: url("http://www.tcpschool.com/examples/images/img_man.png");
background-repeat: no-repeat;
background-position: top right;
}
</style>

이 속성에서 사용할 수 있는 키워드의 조합은 다음과 같다.
1. left top
2. left center
3. left bottom
4. right top
5. right center
6. right bottom
7. center top
8. center center
9. center bottom

또한, 퍼센트(%)나 픽셀(px)을 사용하여 상대 위치를 직접 명시할 수도 있다.
이때 상대 위치를 결정하는 기준은 해당 요소의 왼쪽 상단(left top)이 된다.
https://unsplash.com/ 언스플래쉬 에 접속하여 원하는 아이템을 검색하여 선택된 이미지의 photo ID를 찾은후
htts://source.unsplash.com/{포토ID}/{원하는WIDTHxHEIGHT}입력하면 된다
예를 들어 ttps://unsplash.com/photos/ceITO2rlDgc 여기서 photoID는 ceITO2rlDgc

배경 이미지의 상대 위치를 픽셀 단위로 직접 명시한 예제

<style>
body{
background-image: url("http://www.tcpschool.com/examples/images/img_man.png");
background-repeat: no-repeat;
background-position: 100px 200px;
}
</style>

background-attachment 속성

background-attachment 속성을 사용하여 위치가 설정된 배경 이미지를 해당 위치에 고정시킬 수도 있다.
이렇게 고정된 배경 이미지는 스크롤과는 무관하게 화면의 위치에서 이동하지 않는다.

특정위치에 고정되어 스크롤과 무과하게 이동하지 않는 예제

<style>
body{
background-image: url("http://www.tcpschool.com/examples/images/img_man.png");
background-repeat: no-repeat;
background-position: left bottom;
background-attachment: fixed;
}
</style>

background 속성 한 번에 적용하기

한꺼번에 한줄로 스타일을 설정한 예제

<style>
body { background: #FFCCCC url("http://www.tcpschool.com/examples/images/img_man.png") no-repeat left bottom fixed; }
</style>

CSS background 속성

속성 설명
background 모든 background 속성을 이용한 스타일을 한 줄에 설정할 수 있음.
background-color HTML 요소의 배경색을 설정함.
background-image HTML 요소의 배경 이미지를 설정함.
background-repeat 설정된 배경 이미지의 반복 유무를 설정함.
background-position 반복되지 않는 배경 이미지의 상대 위치를 설정함.
background-attachment 배경 이미지를 스크롤과는 무관하게 해당 위치에 고정시킴.
