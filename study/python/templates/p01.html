표준 입출력 함수

대부분의 프로그램은 사용자의 입력을 받아 동작을 수행한 후 그 결과를 다시 사용자에게 출력하는 형태를 가지고 있다. 따라서 프로그램을 만들 때 입출력 작업은 매우 기본적인 동작이 되며, 대부분의 프로그래밍 언어에서는 표준 입출력 함수라는 것을 통해 이러한 기능을 제공한다.
파이썬에서도 사용자에게 출력 내용을 보여주기 위한 표준 출력 함수로 print() 함수를 제공하며, 사용자의 입력을 받기 위한 표준 입력 함수로 input() 함수를 제공하고 있다.
함수(function)란 하나의 특별한 목적의 작업을 수행하기 위해 독립적으로 설계된 프로그램 코드의 집합으로 정의할 수 있다. 함수를 사용하면 반복적인 프로그래밍을 피할 수 있으며, 모듈화로 인해 전체적인 코드의 가독성이 좋아진다.

print() 함수

파이썬에서는 데이터를 화면에 출력하고 싶을 때 표준 출력 함수인 print() 함수를 사용한다.
자바스크립트에서 console.log과 같은 역할, ()안에 내용을 출력한다

옵션
1. separator option : 옵션 안의 내용을 print안의 요소 사이에 넣어 요소들을 합쳐줌
2. end option : 다음 print함수와 내용을 합쳐서 출력함
3. format option : {}안에 매핑하여 넣어줌

다음 예제는 다양한 타입의 데이터를 print() 함수를 사용하여 출력하는 예제이다.

예제

print(38)
print("문자열")
print([1, 2, 3])

실행 결과
38
문자열
[1, 2, 3]

이와 같이 print() 함수는 숫자나 문자열뿐만 아니라 배열과 같은 데이터까지도 출력할 수 있다.
다음 예제는 여러 데이터를 print() 함수를 사용하여 한 번에 출력하는 예제이다.

예제

print(1+2)
print("파"+"이"+"썬")
print("파""이""썬")
print("파","이","썬")

실행 결과
3
파이썬
파이썬
파 이 썬

파이썬에서는 1번과 2번 라인의 print() 함수처럼 더하기(+) 기호를 사용하여 여러 데이터를 한 번에 출력할 수 있다.
그리고 문자열에 한해 더하기 기호를 생략해도 같은 결과를 출력해 주며, 쉼표(,)를 사용하면 띄어쓰기(white space)를 하나씩 삽입하여 출력한다.

Escape Code 출력

예제

# Escape Code
print('\'you\'') # ''문자로 인식
print('\\you\\\n\n\n') # 줄바꿈
print('\t\t\t\'you\'') # tap

Separator 옵션 출력

예제

# Separator 옵션 : 옵션 안에 문자를 이용해 요소들을 합쳐줌
print('T', 'E', 'S', 'T', sep='') # TEST
print('2019', '02', '19', sep='-') # 2019-02-19
print('abcd', 'google.com', sep='@') # abcd@google.com

print('no end option')

변수가 포함된 문자열 출력

문자열 만드는 방법은 다음 3가지가 있다
1. 서식 지정자(문자열 포맷코드) 사용
2. format 메서드 사용
3. f-string 사용

서식기호 대입 출력

문자열 서식 format 코드

코드설명
%s 문자열(String)
%c 문자 1개(character)
%d 정수(Integer)
%f 부동소수(floating-point)
%o 8진수
%x 16진수
%% Literal % (문자 % 자체)


예제

age = 80
money = 20000
name = 'Kim'
weight = 80.12
etc = 'abcde'
y2 = 'my name is %s, age : %d, weight : %f, money : %d, etc : %s' % (name, age, weight, money, etc)
print(y2)
# my name is Kim, age : 80, weight : 80.120000, money : 20000, etc : abcde

print("%s's favorite number is %d" % ('Jeewon', 7)) # Jeewon's favorite number is 7
print('Test1: %5d, Price: %4.2f' % (776, 6534.123)) # 최대 자리수 정해주기
print("Test1: {0:5d}, Price: {1:4.2f}".format(776, 6534.123)) # 776, Price: 6534.12
print("Test1: {a:5d}, Price: {b:4.2f}".format(a=776, b=6534.123)) # 776, Price: 6534.12

format 함수 사용 문자열 출력

예제

# 문자열 포맷팅
: {}에 들어있는 변수의 값을 문자열 중간 중간에 넣어주기 위해서 사용하는 것.
print('{} and {}'.format('You', 'Me')) # You and Me
print("{0} and {1} and {0}".format('You', 'Me')) # You and Me and You
print("{a} and {b}".format(a='You', b='Me')) # You and Me

# 직접 대입하기
 s1 = 'name : {0}'.format('BlockDMask')
print(s1) # name : BlockDMask

# 변수로 대입 하기
 age = 55
s2 = 'age : {0}'.format(age)
print(s2) # age : 55

# 이름으로 대입하기
 s3 = 'number : {num}, gender : {gen}'.format(num=1234, gen='남')
print(s3) # number : 1234, gender : 남

# 인덱스를 입력하지 않으면?
 s4 = 'name : {}, city : {}'.format('BlockDMask', 'seoul')
print(s4) # name : BlockDMask, city : seoul

# 인덱스 순서가 바뀌면?
 s5 = 'song1 : {1}, song2 : {0}'.format('nunu nana', 'ice cream')
print(s5) # song1 : ice cream, song2 : nunu nana

# 인덱스를 중복해서 입력하면?
 s6 = '\ntest1 : {0}, test2 : {1}, test3 : {0}'.format('인덱스0', '인덱스1')
print(s6) # test1 : 인덱스0, test2 : 인덱스1, test3 : 인덱스0

# 중괄호를 두번 겹치면 중괄호 문자가 나온다
 s7 = 'Format example.{ { } }, {}'.format('test')
print(s7) # Format example.{}, test

# 중괄호로 겹쳐진 인자값
 s8 = 'This is value { { {0} } }'.format(1212)
print(s8) # This is value {1212}

# 왼쪽 정렬
{0:<10} 의미는 {0} 값을 ":<10" 10자리로 표현할건데 왼쪽 정렬을 할 것
# 오른쪽 정렬 {1:>5} 의미는 {1} 값을 ":>5" 5자리로 표현할건데 오른쪽 정렬을 할 것
s9 = '\nthis is {0:<10} | done {1:<5} |'.format('left', 'a')
print(s9) # this is left | done a    |

# 오른쪽 정렬
 s10 = 'this is {0:>10} | done {1:>5} |'.format('right', 'b')
print(s10) # this is right | done     b |

# 가운데 정렬
 s11 = 'this is {0:^10} | done {1:^5} |'.format('center', 'c')
print(s11) # this is center | done   c   |

# 왼쪽 정렬하면서 공백대신 "- /o"로 채우기
 s12 = '\nthis is {0:-<10} | done {1:o<5} |'.format('left', 'a')
print(s12) # this is left------ | done aoooo |

# 오른쪽 정렬하면서 공백대신 "+/0"로 채우기
 s13 = 'this is {0:+>10} | done {1:0>5} |'.format('right', 'b')
print(s13) # this is +++++right | done 0000b |

# 가운데 정렬하면서 공백대신 "./@"로 채우기
 s14 = 'this is {0:.^10} | done {1:@^5} |'.format('center', 'c')
print(s14) # this is ..center.. | done @@c@@ |

# {0:03d} 의미는 {0} 값을 "03d" 정수 3자리로 표현하는데 자리수가 부족하면 0으로 채워진다
# {0:02f} 의미는 {0} 값을 "0.2f" 소수점 2자리까지 표현하고 남는부분은 0으로 채워진다.
# 정수 N자리
 s15 = '\n정수 3자리 : {0:03d}, {1:03d}'.format(12345, 12)
print(s15) # 정수 3자리 : 12345, 012

# 소수점 N자리
 s16 = '아래 2자리 : {0:0.2f}, 아래 5자리 : {1:0.5f}\n'.format(123.1234567, 3.14)
print(s16) # 아래 2자리 : 123.12, 아래 5자리 : 3.14000

# 천단위 구분자
 '{:,}.format(100000000) # 10,000,000
# 백분율
 '{:%}.format(5/12) # 41.666667%
'{:2%}.format(5/12) # 41.67%
# 부호표시
 '{:+f}; {:+f}'.format(3.14 -3.14) # +. - 기호 항상 출력
'{: f}; {: f}'.format(3.14 -3.14) # +는 공백으로, - 기호만 출력
'{:-f}; {:-f}'.format(3.14 -3.14) # - 기호만 출력
# 여러가지 진수표시
 s17 = '정수: {:d}; 16진수: {:#x}; 8진수: {:#o}; 2진수: {:#b}.format(50)
print(s17) # '정수:50; 16진수: 0x32; 8진수: 0o62; 2진수: 0b110010'
# 구구단 표시 (2단과 3단만 표시)
 print('구구단 2단과 3단')
for a in range(2, 4): # 앞에 표시하는 단 수 (2단, 3단)
     for b in range(1, 10): # 뒤에 곱하는 수
        print('{0} x {1} = {2:2}'.format(a, b, a * b))

f-string사용 문자열 출력

f'문자열 {변수} 문자열'
문자열 맨 앞에 f를 붙이고, 출력할 변수, 값을 중괄호 안에 넣는다.

예제

s = 'coffee'
n = 5
result1 = f'저는 {s}를 좋아합니다. 하루 {n}잔 마셔요.'
print(result1) # 저는 coffee를 좋아합니다. 하루 5잔 마셔요.

# f-string 왼쪽 정렬
 x1 = '커피를 left에 두세요'
result1 = f'|{x1:<18}|'
print(result1) # |커피를 left에 두세요      |

# f-string 가운데 정렬
 x2 = '커피를 mid에 두세요'
result2 = f'|{x2:^18}|'
print(result2) # |   커피를 mid에 두세요   |

# f-string 오른쪽 정렬
 x3 = '커피를 right에 두세요'
result3 = f'|{x3:>18}|'
print(result3) # |      커피를 right에 두세요|

# f-string 중괄호 출력
 num = 10
result = f'\nmy age { { { num } } }, { { num } }'
print(result) # my age {10}, {num}

# f-string과 딕셔너리
 d = {'name': 'selfburning', 'gender': 'woman', 'age': 100}
result = f'my name {d["name"]}, gender {d["gender"]}, age {d["age"]}'
print(result) # my name selfburning, gender woman, age 100

# f-string과 리스트
 n = [100, 200, 300]
print(f'\nlist : {n[0]}, {n[1]}, {n[2]}') # list : 100, 200, 300

for v in n:
print(f'list with for : {v}')
# list with for : 100
# list with for : 200
# list with for : 300

input() 함수

파이썬에서는 프로그램 실행 도중 사용자의 입력을 전달받아 그 값을 프로그램에서 사용하고 싶을 때 input() 함수를 사용할 수 있다.
다음 예제는 사용자가 입력한 데이터를 곧바로 출력하는 예제이다.

예제

var = input()
print(var)
하지만 위의 예제와 같은 경우에는 사용자 입장에서 현재 프로그램이 자신의 입력을 받기 위해 정지해 있는 건지 아니면 다른 이유로 정지된 건지를 명확히 알 수 없다.
따라서 다음 예제와 같이 입력 안내 문자열을 input() 함수에 인수로 전달하여 사용자에게 입력에 관한 정보나 입력을 해달라는 요청 메시지를 전달하는 것이 좋다.
예제

var = input("출력할 문자열을 입력해 주세요 : ")
print(var)

주석(comments)

주석(comments)이란 해당 프로그램 소스 코드를 이해하는데 도움을 주는 설명을 적거나 개발자가 디버깅을 위해 작성한 코드를 가리킨다.
이러한 주석은 파이썬 인터프리터에 의해 해석되지 않으므로 실행되지 않는다.
파이썬에서는 해시(#) 기호를 사용하여 주석을 작성할 수 있다.

예제

# 작성자 : kjyschool
# 수정날짜 : 2023.05.15
print(38)# 숫자를 출력하는 코드
print("문자열")# 문자열을 출력하는 코드
print([1, 2, 3]) # 리스트를 출력하는 코드

실행 결과
38
문자열
[1, 2, 3]

위의 실행결과를 살펴보면 주석으로 처리된 문자열은 프로그램에 어떠한 영향도 미치지 않음을 확인할 수 있다.
위 예제의 1~2번 라인의 주석처럼 여러 라인에 걸쳐 작성해야 하는 주석은 다음과 같이 세 개의 큰따옴표(""")나 작은따옴표(''')를 사용하여 더욱 손쉽게 작성할 수 있다.
