HTML Emmets

Emmet이란?

강력한 자동완성 기능 등으로 HTML 작성 속도를 크게 향상시켜주는 플러그인

Emmet 문법

1. Tab : 요소 (Elements) 생성

생성하려는 요소의 이름을 입력한 뒤 tab을 누르면 태그가 자동 생성된다.
Emmet에는 미리 정해진 태그 이름이 없다. 즉, 아무 이름으로나 태그를 생성할 수 있다.(에디터에 따라 그렇지 않은 경우도 있다.)

예제> html:5[tab]

결과

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">
<title>Document</title>
</head>
<body>
</body>
</html>

예제> helloworld[tab]

결과

<helloworld></helloworld>

구조화 하기 (Nesting operators)

Emmet의 문법을 활용하여 요소들의 구조를 간편히 생성할 수 있다.

2. > : 자식 요소. 중첩

>를 사용하여 자식 요소를 생성할 수 있다.

예제) div>ul>li>ul

결과

<div>
<ul>
<li></li>
<ul>

3. + : 형제 요소

+ 를 사용하여 한 요소와 같은 단계에 위치한 요소를 생성할 수 있다.

예제) div+p+bq

결과

<div></div>
<p></p>
<blockquote></blockquote>

4. ^ : 한 단계 올리기

^ 를 사용하여 한 단계 위에 요소를 배치할 수 있다.

예제) div+div>p>span+em^bq
span+em은 p태그의 하위 단계에 위치하지만 bq는 ^로 인해 p태그와 같은 단계에 위치하게 된다.

결과

<div></div>
<div>
   <p>
      <span></span>
      <em></em>
   </p>
   <blockquote></blockquote>
</div>

^ 를 사용한 만큼 윗단계로 올라가 위치한다

예제) div+div>p>span+em^bq

결과

<div></div>
<div>
   <p>
      <span></span>
      <em></em>
   </p>
   <blockquote></blockquote>
</div>

예제) div+div>p>span+em^^bq

결과

<div></div>
<div>
   <p>
      <span></span>
      <em></em>
   </p>
<blockquote></blockquote>
</div>

예제) div+div>p>span+em^^^bq

결과

   <div></div>
   <div>
      <p>
         <span></span>
         <em></em>
      </p>
<blockquote></blockquote>
   </div>

5. 요소 * : 곱하기. 반복하기

특정요소를 반복생성 하고 싶은 개수 만큼 * 를 입력한다.

예제) ul>li*5

결과

<ul>
<li></li>
<li></li>
<li></li>
<li></li>
<li></li>
</ul>

예제) ul>li*3

결과

<ul>
<li></li>
<li></li>
<li></li>
</ul>

6. (): 그룹화

복잡한 축약어들을 () 로 하위트리로 그룹화한다.

예제) div>(header>ul>li*2>a)+footer>p

결과

<div>
   <header>
      <ul>
         <li><a href=""></a></li>
         <li><a href=""></a></li>
      </ul>
   </header>
   <footer>
      <p></p>
   </footer>
</div>

각 그룹은 축약어 하위 트리를 포함하고 이후 모든 요소는 그룹의 첫 번째 요소와 동일한 수준으로 삽입된다.
또한 * 를 이용해 이웃한 그룹끼리 결합도 가능하다

예제) (div>dl>(dt+dd)*3)+footer>p

결과

<div>
   <dl>
      <dt></dt>
      <dd></dd>
      <dt></dt>
      <dd></dd>
      <dt></dt>
      <dd></dd>
   </dl>
</div>
<footer>
   <p></p>
</footer>

7. ".", "#" : 속성 연산자 attribute operators (ID and CLASS)

특정 ID 의 단축키는 #id로 , 특정 CLASS의 단축키는 . 으로 이용한다

예제) div#header+div.page+div#footer.class1.class2.class3

결과

<div id="header"></div>
<div class="page"></div>
<div id="footer" class="class1 class2 class3"></div>

8. []: 관습 속성 custom attributes

요소에 [] 을 이용하여 연관된 관습 속성을 입력한다.
<ul>나 <li> 등을 여러개 반복했을때 </ul>나 </li> 를 일일히 입력할 필요 없다.
내용에 빈칸이 없다면 큰(작은) 따옴표도 입력할 필요 없다

예제) td[title="Hello world!" colspan=3]

결과

<td title="Hello world!" colspan="3"></td>

9. $: 아이템 번호 매기기

* 는 요소를 반복할수 있고 $ 는 요소에 번호(순서)를 매길수 있다.
요소 이름, 속성 이름 또는 속성 값 안에 $ 연산자를 입력하여 현재 반복되는 요소 번호를 생성한다.

예제) ul>li.item$*5

결과

<ul>
<li class="item1"></li>
<li class="item2"></li>
<li class="item3"></li>
<li class="item4"></li>
<li class="item5"></li>
</ul>

예제) ul>li.item$$$*4

결과

<ul>
<li class="item001"></li>
<li class="item002"></li>
<li class="item003"></li>
<li class="item004"></li>
</ul>

10. $@: 번호 방향 바꾸어 매기기 오름순 내림순

예제) ul>li.item$@-*5

결과

<ul>
<li class="item5"></li>
<li class="item4"></li>
<li class="item3"></li>
<li class="item2"></li>
<li class="item1"></li>
</ul>

11. @N: 시작번호 바꾸어 매기기

예제) ul>li.item$@2*4

결과

<ul>
<li class="item2"></li>
<li class="item3"></li>
<li class="item4"></li>
<li class="item5"></li>
</ul>

예제) ul>li.item$@-3*4

결과

<ul>
<li class="item6"></li>
<li class="item5"></li>
<li class="item4"></li>
<li class="item3"></li>
</ul>

12. {}: 텍스트 추가하기

예제) a{Click me}

결과

<a href="">Click me</a>

{text}가 요소 바로다음에 입력되면 부모 텍스트는 변경되지 않는다.

예제) p>{Click}+a{here}+{to continue}

결과

<p>Click</p>
<a href="">here</a>to continue
