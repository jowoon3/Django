HTML 리스트(List)

리스트(list)란 여러 요소들을 일렬로 나열한 목록이나 명단을 말한다.

1. 순서가 없는 리스트(Unordered List)
2. 순서가 있는 리스트(Ordered List)
3. 정의 리스트(Definition List)

순서가 없는 리스트(UL)

순서가 없는 리스트는 <ul>태그로 시작하며, 여기에 포함되는 각각의 리스트 요소는 <li>태그로 시작한다.

각각의 리스트 요소 앞에는 기본 마커(marker)로 검정색의 작은 원(bullet)이 위치한다.

  • 사과
  • 멜론
  • 바나나

CSS의 list-style-type 속성을 사용하면 리스트 요소 앞에 위치하는 마커(marker)를 다른 모양으로 변경할 수 있다.

- disc : 검정색 작은 원 모양 (기본설정)
- circle : 흰색 작은 원 모양
- square : 사각형 모양

  • 수박
  • 참외
  • 옥수수

순서가 있는 리스트(OL)

순서가 있는 리스트는 <ol>태그로 시작하며, 여기에 포함되는 각각의 리스트 요소는 <li>태그로 시작한다.
각각의 리스트 요소 앞에는 기본 마커로 아라비아 숫자가 위치한다.

  1. 사과
  2. 멜론
  3. 바나나

CSS의 list-style-type 속성을 사용하면 리스트 요소 앞에 위치하는 마커(marker)를 다른 모양으로 변경할 수 있다.

- decimal : 숫자 (기본설정)
- upper-alpha : 영문 대문자
- lower-alpha : 영문 소문자
- upper-roman : 로마 숫자 대문자
- lower-roman : 로마 숫자 소문자

  1. 수박
  2. 참외
  3. 옥수수

정의 리스트(description list)

정의 리스트(description list)는 용어와 그에 대한 정의를 모아놓은 리스트로 <dl>태그로 시작한다.

<dt>태그에는 용어의 이름이 들어가고, <dd>태그에는 해당 용어에 대한 정의를 기술한다.

상추

HTML 테이블(Table)

테이블(Table)이란 여러 종류의 데이터(data)를 보기 좋게 정리하여 보여주는 표이다.

HTML에서는 <table>태그를 사용하여 표를 작성한다.

1. <tr>태그는 테이블에서 열을 구분해 준다. table row
2. <th>태그는각 열의 제목을 나타내며, 모든 내용은 자동으로 굵은 글씨에 가운데 정렬이 된다. table header cells
3. <td>태그는 테이블의 열을 각각의 셀(cell)로 나누어 준다. table divide

순서는 table > tr (첫째행) > 제목줄인 첫째행의 열나눔 th > tr (둘째행) > 열구분해서 데이터입력 td
보통 tr(행)은 배경스타일을 설정하고 th와 td (열)은 글자 스타일과 경계선을 설정한다.

CSS의 border 속성을 이용하여 테이블에 테두리를 표현할 수 있다.
border 속성값을 따로 명시하지 않으면, 해당 테이블은 빈 테두리를 가지게 된다

예제

테이블의 테두리(border)가 두 줄씩 나타나는 이유는 <table>태그와 <th>태그, <td>태그가 모두 자신만의 테두리를 가지고 있기 때문이다.
위와 같이 두 줄로 표현되는 테두리를 한 줄로 설정하려면 border-collapse 속성값을 collapse로 설정하면 된다
