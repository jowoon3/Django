HTML 기본 구조

Head는 HTML 문서의 메타데이터(metadata)를 정의한다.
메타데이터(metadata)란 HTML 문서에 대한 정보(data)로 브라우저에는 표현되지 않는다
메타데이터는 title, style, meta, link, script, base태그 등을 이용하여 표현

HTML 요소 element 구조

HTML 요소(element)는 여러 속성(attribute)을 가진다.

HTML 요소는 시작 태그로 시작해서 종료 태그로 끝난다.

>

속성은 HTML 요소 중에서도 언제나 시작 태그 내에서만 정의되며, 속성 이름과 속성값(value)으로 표현한다.

속성 이름은 언제나 소문자로 작성!

HTML5 표준은 대소문자를 구분하지 않음

W3C에서는 속성 이름을 소문자로 작성하도록 권장

XHTML에서는 속성 이름을 더욱 엄격하게 소문자로만 사용

속성값은 언제나 따옴표로 감싼다!! (작은따옴표도 가능)

HTML5 표준에서는 따옴표 사용을 강제하지 않지만 오류발생 가능
