람다 표현식(lambda expression)이란?

람다 표현식(lambda expression)이란 간단히 말해 메소드를 하나의 식으로 표현한 것이다.

함수를 람다 표현식으로 표현하기

lambda 매개변수(들): 식
(lambda 매개변수(들): 식)(인수(들))

먼저 숫자를 받은 뒤 10을 더해서 반환하는 함수 plus_ten을 만들어보자

예제

>>> def plus_ten(x):
... return x + 10
...
>>> plus_ten(1) # 11
---------- 람다표현식 ----------
>>> lambda x: x + 10
# <function <lambda> at 0x02C27270>
// 실행결과 함수 객체가 나오는데, 이 상태로는 함수를 호출할 수 없다. 람다 표현식을 익명 함수(anonymous function, 이름이 없는 함수)라고 한다

---------- 람다표현식 호출 ----------
>>> plus_ten = lambda x: x + 10 // 익명함수인 람다 표현식을 호출하려면 변수에 할당해줘야 한다
>>> plus_ten(1) # 11

---------- 람다표현식 호출 ----------
>>> lambda x: x + 10(1) // # 11 매개변수에 할당하지 않고 괄호안에 (람다 표현식)을 넣고 바로 (인수)를 대입해줘도 된다

주의 : 람다표현식 안에서는 변수를 만들 수 없다

람다 표현식에서 주의할 점은 람다 표현식 안에서는 새 변수를 만들 수 없다는 점이다.
반환값 부분은 변수 없이 식 한 줄로 표현할 수 있어야 한다.
만약 변수가 필요한 코드일 경우에는 def로 함수를 작성하는 것이 좋다.

예제

>>> (lambda x: y = 10; x + y)(1)
SyntaxError: invalid syntax

단, 람다 표현식 바깥에 있는 변수는 사용할 수 있다.

예제

>>> y = 10
>>> (lambda x: x + y)(1) # 11

람다 표현식을 인수로 사용하기

람다 표현식을 사용하는 이유는 함수의 인수 부분에서 간단하게 함수를 만들기 위해서다. 이런 방식으로 사용하는 대표적인 예가 map이다.

예제

---------- 함수를 만들어 map 사용하기 ----------
>>> def plus_ten(x):
... return x + 10
...
>>> list(map(plus_ten, [1, 2, 3])) # [11, 12, 13]

---------- 람다표현식으로 함수를 만들어 map 사용하기 ----------
>>> list(map(lambda x: x + 10, [1, 2, 3])) # [11, 12, 13]
